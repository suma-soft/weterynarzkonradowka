# Gabinet Weterynaryjny Katarzyna Suchecka-Kućko

## Opis
Strona internetowa Gabinetu Weterynaryjnego lek. wet. Katarzyny Sucheckiej-Kućko zlokalizowanego w Konradówce, Chojnów.

## Oferowane usługi
- **Profilaktyka**: Szczepienia ochronne, zwalczanie pasożytów wewnętrznych i zewnętrznych
- **Diagnostyka i leczenie**: Leczenie chorób wewnętrznych, badanie moczu, badanie krwi (w laboratorium zewnętrznym), zeskrobiny i rozmazy cytologiczne
- **Zabiegi chirurgiczne**: Kastracje, sterylizacje, drobne zabiegi chirurgiczne
- **Pielęgnacja**: Czyszczenie uszu, obcinanie pazurków, usuwanie kamienia nazębnego metodą ultradźwiękową
- **Inne**: Czipowanie zwierząt wraz z wprowadzeniem do bazy danych, zakładanie paszportów

## Zastosowane technologie
- HTML5
- CSS3 (Bootstrap 4)
- JavaScript (jQuery)
- Ikony Font Awesome
- Responsywny design
- Preprocesor SCSS
- System budowania Gulp

## Funkcjonalności
- Responsywny design mobile-first
- Interaktywne karty usług z animacjami flip
- Informacje kontaktowe i mapa lokalizacji
- Animacje ładowania
- Profesjonalna oś czasu przedstawiająca historię gabinetu
- Integracja z mediami społecznościowymi

## Instalacja
1. Sklonuj repozytorium
2. Zainstaluj zależności: `npm install`
3. Uruchom serwer deweloperski: `gulp`
